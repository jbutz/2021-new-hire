Create a class to represent an individual bank account. In order to create an instance of this class you must provide:

  • The account owner's name
  • The account's starting balance
  • The account's currency. Valid values are ABC or XYZ

The bank account class must support the following operations:

  • Deposits, accepting a positive number to add to the account balance
  • Withdrawals, accepting a positive number to subtract from the account balance
  • Current exchanges, special withdrawals to a different currency

Make all of your changes in the script.js file. Heed the comments at the top of the file. A function has already been created that calls an API to perform the currency conversions.

The function either returns an object with from, to, and rate properties, or an object with only an error property.

Once you have your class created, create a new instance of the class with an initial balance of 100 ABC. Then deposit 350 ABC. Then perform a currency exchange to XYZ and return that value from the run() function.
